In Minecraft you can use bone meal on tall flowers to replicate them, why not do the same with small flowers?

<p align="center">
  <img src="https://i.imgur.com/oOnz4V2.gif">
</p>

## Usage
Use bone meal on a small flower to get more!

## Config
```yaml
# Whether wither roses should be excluded from replication
excludeWitherRose: true
```

## Commands
**/botanical info** - Plugin version

**/botanical reload** - Reload config (OP)

## Permissions
botanical.reload (OP)
